Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language.
  • HTML is the standard markup language for creating web pages.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS describes how HTML elements are to be displayed on screen,paper or in other media documents.
  • CSS saves a lot of work. It can control the layout of multiple web pages at once.
  • External stylesheets are stored in CSS files.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git helps you keep track of code changes.

JavaScript

  • JavaScript is a programming language that enables developers to create and construct multiple web pages features.
